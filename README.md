# Insightful Data Hub

A professional, dashboard-style analytics platform for data analysts, students, and business users to upload raw datasets, assess data quality, clean issues, and generate insights.

## Features

- **Data Upload**: Support for CSV, Excel (.xlsx), and PDF files.
- **Data Quality Overview**: Interactive table previews and health indicators.
- **Data Cleaning**: Tools to handle duplicates, missing values, and normalization.
- **Analysis Dashboard**: Automated charts and summary insights.
- **Reports**: Export cleaned data and PDF reports.

## Local Development

1. **Install dependencies**:
   ```sh
   npm install
   ```

2. **Run development server**:
   ```sh
   npm run dev
   ```
